[![Build Status](https://travis-ci.org/PointKernel/tensor-core-miniapp.svg?branch=master)](https://travis-ci.org/PointKernel/tensor-core-miniapp)

Mini-apps for CUDA measurement

## Build
```bash
make
```

## Run
To measure Tensor Core FLOPs:
```bash
./simpleGEMM
```

To measure overhead:
```bash
./simpleOverhead
```
